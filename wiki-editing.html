wiki editing

As said before, we have two different ways of editing the wiki:

  1. through github.com gollum web interface
  2. locally, as with any other git repository

There are two things which cannot be done through 1. adding folders and creating sidebars. With respect to folders, AFAIK it is also impossible to create a new page in a folder different from the top one through 1.

Let's see then how we can setup our local copy of the wiki; once you're used to it, you'll probably prefer local editing, at least for big editings, re-organizing stuff, etc.

local editing setup and how-to

This does not differ a lot from standard github repo setup, but anyway let's give a detailed walkthrough.

clone the wiki repo

Inside your repos folder on your local machine, 

    git clone git@github.com:wiki-ohnosequences/wiki-ohnosequences.github.com.wiki.git

that's all! :)

editing

cd into your repo, and grab the latest version

    cd wiki-ohnosequences.github.com.wiki
    # if you're feeling lucky, just pull
    git fetch

Now, just add/change whatever you want, and when you're done

    git add <whatever>
    git commit -m 'awesome docs added'

The only thing left is pushing

    git push
